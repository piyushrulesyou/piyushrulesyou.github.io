PathLabs
This project was made under TCS Internship.

It is a Web Based Application, through which a customer can register for a Pathology test from home as well as can check and download the test reports from home. Also they can keep a track on their health issues. The frontend of this project is made using HTML, CSS, Bootstrap, JQuery and JavaScript. For the backend and database JAVA is used. In Java, Servlet, JDBC, JSP is used. MySQL Database is used to store the details of customers and the tests being conducted. The project is made using MVC (Model View Controller) Architecture.

Blood Box
This Project is a step towards saving the life of people who are in immediate need of blood.

An Online Portal whose main aim is to help those peoples who are in immediate need of blood. The person willing to donate blood can upload his/her details. The person in need can then search for the required blood group and his present location. If donors are present then the contact details and the address of the donors will be provided to the receiver. The frontend of this project is made using GWT (Google Web Toolkit.) For the backend JDBC and MySQL Database is used. RPC (Remote Procedural Call) Architecture is used.
